# Steam物品类型数据库
每日自动更新Steam中所有公开AppID类型数据，北京时间早晚八点半更新一次

## 说明

全类型数据 `data/output.json`

精简版数据(仅包含DLC及原声带) `data/Listwithnogame.json`
